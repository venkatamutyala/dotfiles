# My dev setup



## Get started with:

```bash
curl setup.venkatamutyala.com | zsh
```


## MacOS Setup:

- Install google chrome
- Install tailscale
- Install 1password (configure SSH agent in 1password)
- Speed up mouse sensitivity to max settings
- Confirm machine is encrypted (file vault should be enabled by default).
- Do not connect icloud/etc. Keep it entirely offline. Do not save disk recovery keys.
